【艺术在线】一对耳环破纪录拍出4亿人民币 彩钻拍卖要逆天

来源：admin     发布时间：2018-09-12

      近年来，在世界拍卖市场上，大尺寸且颜色好、净度高的宝石和钻石尤其受到追捧，彩钻屡屡成为近年来拍卖中的“黑马”拍品，落槌价更是接连打破拍卖纪录。

      上周举行的日内瓦苏富比“瑰丽珠宝及贵族首饰”拍卖共刷新三项世界拍卖纪录。“阿波罗与阿特米斯”钻石合共拍出57,425,478 美元（约4亿人民币），刷新耳环世界拍卖纪录，并被命名为“The Memory of Autumn Leaves”及“The Dream of Autumn Leaves”。

      “阿波罗与阿特米斯”钻石，合共拍出57,425,478 美元（约4亿人民币），刷新耳环世界拍卖纪录。

      其中被命名为“The Memory of Autumn Leaves”的耳环镶嵌了一枚重达14.54克拉艳彩蓝钻，其成交价为42,087,302美元（约3.2亿人民币），而被命名为“The Dream of Autumn Leaves”浓彩粉红钻重达16.00克拉，拍出了15,338,176美元（约1亿人民币）的价格。

据悉，近年来高克拉数的彩色钻石在拍卖场上备受追捧，是增值最快的拍场宝石。同场拍卖中，一枚伯爵（Piaget）7.04克拉浓彩紫粉红色钻石戒指也拍出高价，以13,245,750美元（约9100万人民币），每克拉的成交价达到1,881,499美元（约1296万人民币）。这枚戒指刷新了浓彩紫粉红色钻石世界拍卖纪录，同时也刷新了浓彩紫粉红钻石每克拉价格世界拍卖纪录。

      10.10克拉内部无瑕椭圆形鲜彩蓝钻“迪比尔斯千禧瑰宝4”在2016年香港苏富比春拍上以3,180万美元成交，刷新当时亚洲珠宝拍卖单价纪录。

      2016年出现在苏富比日内瓦的15.38克拉鲜彩粉红钻——“无双”粉红钻。

      在每年开采的数以百万计的钻石中，仅0.001%的钻石符合彩钻的标准，而这其中只有为数不多的彩钻可达到顶级的浓彩和艳彩的等级。专家指出，色彩在钻石的价值中起着关键作用，彩钻珍贵的原因之一源自其稀有性，红色钻石几乎微乎其微，绿色也极罕见，蓝色、紫色都是稀有的颜色，比较起来，黄色、棕色等颜色相对较多，但整体而言，彩钻仍是稀少的宝石，具有一定的收藏价值。

      铂金镶嵌26.65克拉艳彩黄钻及钻石戒指，来自蒂芙尼2017 BlueBook“丛林幻境”高级珠宝系列。

      为什么拍卖行是人们可以发现顶级彩钻的唯一场所？美国宝石研究院(GIA)的罗素·肖尔在接受媒体采访时表示，这一看似简单的结论起因于拍卖的悠久历史、对全球拍卖趋势的良好掌控、善于发现稀有宝石的意识，以及与国际知名拍卖行长期积累的人脉。他说：“首先，日内瓦、纽约和香港三大主要市场总能吸引全球客户。罕见的彩色钻石总能在这些拍卖行里卖出史无前例的价格。其次，苏富比和佳士得对顶级的稀有钻石有最深刻的洞察力，换句话说，货真价实的珍贵钻石在这里流通。最后，私人买家的人数随之增加。”

      一颗出众的彩色钻石非常罕见，其质量由颜色的浓度和色彩的吸引力决定。与鉴定无色钻石的4C标准不同，鉴定机构对彩色钻石的鉴定采用一套独特的系统。罗素·肖尔表示：“鉴定从色彩开始 ，例如，首先鉴定钻石为黄色，接下来，如果钻石中有一点儿绿色的混色，这颗钻石则被称作‘黄中带绿’的绿黄色钻石。然后鉴定颜色的浓度，从微到浓彩，艳彩等。GIA也会鉴定颜色是否为天然。”
