【艺术在线】惊爆！温都水城康熙行宫惊现估值数十亿宝玉原石

来源：admin     发布时间：2018-09-13

      8月24日，由北京宏福集团、缅甸美景矿业集团有限公司和温都水城康熙行宫玉文化博物馆联合主办，主题为“鉴赏行宫玉翠，品味皇城文韵”的珍稀文玩收藏展在位于温都水城旅游景区的康熙行宫内隆重开展。其中单品估值数亿的翡翠原石，估计二十多亿的红宝石原石惊爆现场。本次展览共展出上千件的藏品，其中有200余件原石，数件单品价值千万元；数十件摆件，包括当代著名玉工艺美术大师宋世义等人的代表名作；几千件成品，其中不乏单价数亿甚至几十亿的稀世珍品。除展出玉石翡翠展品，还有乌木雕刻工艺品、康雍乾朝古董同时随展展出。展览期间，著名玉器研究专家、缅甸美景矿业公司董事长杨银山为广大参会人员讲解玉文化相关的知识，分享玉器鉴定的经验体会，介绍三块精品原石并与北京宏福集团董事局主席黄福水先生、中国宝玉石协会洪贵明先生共同切割一块价值上亿原石并开光揭彩。整个专业详细的原石“开门”过程在切割房为到场嘉宾展示。北京玉器文化源远流长，深厚精湛。明清两代是玉器发展的高峰时期，作为都城的北京汇集了全国各地的玉作巧匠，因而北京玉器融南、北玉作之美，集两家之长，形成北京独特之风格。同时大量质优物美的玉器涌入京城，使得北京成为著名的玉器流通枢纽。此次在历史底蕴深厚的皇城旧址——康熙行宫举办的玉器珍玩展览，既与玉文化古朴悠久的风格一脉相承，展示了京城玉器的汇集大成，又激活了这座昔日行宫别院的皇家气象，尽显皇城文韵。

      据悉，此次展览展出的藏品仅为一部分，未来在康熙行宫还会有更多玉石珍品、文玩古董不定期进行展出，为广大玉器、文玩爱好者和相关从业人员提供一个稳定优质的交流平台。
