【艺术在线】艺术品收藏：别用巨资收藏“无知”

来源：admin     发布时间：2018-09-12

　　艺术品收藏是一个很老的话题。自古至今有数不尽的藏家，他们用收藏的作品物证了中国的艺术史。

　　文化艺术是文明之公器。表面上看，艺术品的收藏纯粹属于个体的行为，但是决定其内在价值的因素，则是在于时间的考验、艺术的规律，不属于收藏者。因此，收藏者的个人判断和爱好只有符合历史的发展、艺术自身的内在规律，才可能使他收藏的作品具有真正的审美价值和投资前景，否则无论收藏者如何敝帚自珍，都无法兑现他的愿望。

　　面对纷繁复杂、琳琅满目的现当代艺术品市场，藏家要考虑的是收藏哪一类的作品。毫无疑问，除已经定论的、已故的古典大师的作品以外，一定要收藏现代形态的艺术，只有现代艺术才能反映现代生活，传统形态的艺术与现代文明已经脱节或背离;而一些画家试图用传统法则创作现代艺术无异于缘木求鱼，这是艺术发展中残酷的规律之一。

　　时代的变迁，社会的高速发展以及全球文化的广泛交流，极大地影响了中国画家的创造，也影响了艺术品的收藏。但是，许多收藏者的眼光落后于前卫的创造者，从而导致许多创造性的艺术家被收藏家忽视或者遗忘，而因袭守旧的平庸之作却备受青睐。更为致命的是，人们对传统艺术的欣赏已经有了深厚的积淀，对传统类艺术的理解、收藏都是一件顺理成章的事情。但对于从内在的审美到外在的形式都发生了巨大变化的现代艺术，缺乏正确的认识和有效的判断。时至今日，仍然有人否认林风眠、吴冠中的作品是中国画。一些收藏者因为在这方面的无知而粗暴地对待现代艺术，简单地用传统标准来衡量现代艺术，进而误认为现代艺术是乱涂乱画、毫无道理。

　　对于一个普通的收藏者而言，收藏的目的基本上是两个方面：第一是满足自己对艺术审美的需要，收藏自己认为美的、有价值的作品，用于装饰自己的生活空间，享受艺术作品带来的精神愉悦;第二是希望通过在艺术品上的投资，经过一定的时间或者操作，达到增值的目的，增加财富。当然两个目的常常兼而有之。在这两个目的之中，第一个目的是至关重要的，只有美的、有价值的艺术品才能美化我们的生活，丰富与提升精神生活，实现艺术品的终极意义。而且只有实现了第一个目的才有可能实现第二个目的，因为没有审美价值的艺术品是不可能增值的。因此，准确判断收藏品的审美价值是收藏的核心。

　　关于审美问题，可以说是众说纷纭，莫衷一是，这局面是由审美本身的丰富性和多样性决定的。正因为如此，每一个收藏家都认为自己收藏的作品是最有审美价值的。但是，原来许多投机一时，价格暴涨的名家、书画官员的字画，如今都成了烫手山芋，这是很残酷的，无法躲避的现实。这些作品既没有审美价值，更谈不上增值的可能。事实证明，收藏这类艺术品的藏家，是用巨资收藏了自己的愚蠢和无知。

　　是什么导致他们在审美判断上产生严重的误差?虽然审美判断是一个极其复杂的精神劳动，在这过程中都有可能产生各种错误，其中最严重的错误是收藏者理不清艺术品背后一些最基本的关系。

　　第一，许多藏家满足于视觉上世俗性的愉悦。艺术品不是简单的视觉美否，它的本质属性是文化的、精神的、时代的。如果仅仅从好看的层面去要求，那么，千姿百态的美女像和五颜六色的风景照片，足以取代所有的艺术品。仅仅从好看的角度去收藏显然是错误的，而应该从作品内在的文化、精神及时代层面去考量，寻找作品中的一些含义以及审美趣味等等。但是，在时代性这个问题上，收藏者往往无法把握，难以判断，以至常常忽略。而时代性对艺术作品是极为重要的，艺术作品以它特有的方式反映了人类文明发展的一个过程，失去了时代性的艺术作品将失去它最根本的职能。我们这个时代区别于传统农耕文明，这种区别不是量的区别，而是质的区别，这种区别导致了艺术作品的价值取向、审美判断等重要方面都发生了本质性的转变。而当今许多中国画仍然停留在农耕文明的审美状态之中，在价值取向上也没有走出传统文化的范畴。所以，这类以传统面貌表达的艺术作品，从本质上来说是没有意义的，因为它们与我们的时代和社会没有关系，它们仅仅是对传统艺术外表的语言形式上的模仿和继承;没有我们这个时代的精神面貌，是失血断气的作品。艺术作品应该体现出人类现代文明中具有普遍意义的精神食粮，传统文化中倡导的归隐、无为、避世这类封闭、狭隘的价值理念都应该退出现代艺术作品。这样，中国画才能具有世界性，成为人类共同的精神财富。

　　第二，对现代艺术作品而言，至关重要的一点在于，其外在的语言形式必须具有独创性。那些沉湎于传统的收藏者，他们对中国画语言的判断力也局限于传统笔墨范畴。他们没有能力欣赏现代型的中国画，并用粗暴的态度拒绝。这个拒绝本质上是对艺术的无知，也断送了他收藏的未来。

　　现当代中国画的重要贡献之一，就是对传统笔墨的反叛，因为现代艺术强调个体价值的表达，每一个艺术家都要以最大的努力创造出属于自己的艺术语言，雷同于传统、他人都是一种不同程度的失败。这个要求对任何一个现当代艺术家而言都是极其严酷的，这是综合考验和检测一个艺术家的素质、天赋、技能等的有效方法，其难度是一个传统型的中国画家难以望其项背的。

　　因此，收藏现代型中国画是收藏中国画的华山道，要求收藏者重新学习，重建知识结构，识别现代形态中国画的高低贵贱，虽然困难重重，但这是不二法门。
