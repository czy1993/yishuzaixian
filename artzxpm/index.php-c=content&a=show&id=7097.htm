故宫规模最大的院外展览亮相太原

来源：admin     发布时间：2018-10-04

　　10月1日，“2018太原·故宫文物展”在太原博物馆开幕，故宫博物院七项展览亮相山西太原，成为故宫博物院历史上规模最大的一次院外展览。山西省委副书记、省长楼阳生、故宫博物院院长单霁翔等出席开幕式并发表致辞。而举办展览的太原博物馆位于汾河岸边的长风文化商务区中，馆舍建筑由5个相对独立又相互连通的椭圆倒锥体块连接组合而成，是保罗·安德鲁巴黎建筑师事务所和北京市建筑设计研究院三所共同设计的作品。

展览现场

　　本次展览从2018年10月1日开始，至2019年3月31日结束，历时半年，共有7个专题展。分别为《普天同庆——清代万寿盛典展》、《凝华焕彩——故宫博物院藏宫廷珐琅器精品展》、《御瓷撷英——明代御窑瓷器展》、《笔走龙蛇——傅山及明末清初名家行草书特展》、《日丽紫禁——明清官式建筑展》、《紫禁藏影——故宫博物院藏老照片展》和《数字故宫·万象紫禁》。

　　其中，《普天同庆——清代万寿盛典展》规模最大，展示乾隆、慈禧寿庆的礼仪形式、筵宴情况。展厅里复原了部分“畅音阁”戏楼，也展示了大型的钟、磬。唐玄宗始以皇帝诞辰称千秋节，此后历代相沿而名称不同。清代皇帝寿辰称万寿节，皇太后、皇后寿辰分别称圣寿节、千秋节。万寿节成为清宫与元旦、冬至并列的三大节日之一。清代帝后倡导孝悌之风，尤重逢旬大庆，甲子（60岁）、古稀（70岁）或更高寿，均要隆庆，现场的展陈也处处体现着“寿”与“喜”的主题。

清铜鎏金无量寿佛

畅音阁戏楼的结构示意图

　　《凝华焕彩——故宫博物院藏宫廷珐琅器精品展》遴选150件具有代表性的珐琅器，按照典章、生活、陈设、宗教等功能展示，力求全方位反映宫廷珐琅器的整体面貌和帝王的审美意趣。

　　珐琅是舶来工艺，自元代传入后，经过吸收、改造，逐渐实现本土化，成为我国工艺门类之一。明、清两代，尤其受到宫廷的重视，因此得以保持上乘品质，持续发展，并影响至今。在我国，以金属作胎，采用珐琅做器表装饰的工艺统称为珐琅工艺，主要种类有掐丝珐琅、画珐琅、透明珐琅、内填珐琅等。其中掐丝珐琅和画珐琅工艺被广泛采用，二者发源地不同、流传有别、制造方法和风格面貌有异，堪称双峰并峙。

掐丝珐琅明皇试马图挂屏

　　本展览从故宫博物院收藏的珐琅器中遴选出150件具有代表性的作品予以展示，按功能分为典章、宗教、生活、陈设等四部分，全方位展示宫廷珐琅器的整体面貌和帝王的审美意趣。

掐丝珐琅七珍

　　《御瓷撷英——明代御窑瓷器展》从故宫博物院所藏瓷器中遴选出150件自明代洪武至万历各朝具有代表性的御窑瓷器予以展示，旨在为观众朋友提供一个较全面了解明代景德镇御窑烧造瓷器品种、所取得艺术成就和欣赏标准器的机会。

《御瓷撷英——明代御窑瓷器展》展区

　　明代宫廷无论在日常陈设、祭祀、宗教或内外赏赐等方面，都需要使用大量的陶瓷器。为满足这些需求，明代朝廷除委派地方民窑烧造和接受民窑进贡外，还在景德镇设“陶厂”（建文四年更名为“御器厂”），专门烧造宫廷用瓷。终明一代，除个别时期曾因故减烧或停烧之外，御器厂为宫廷烧造了大量瓷器。故宫博物院是目前收藏明代御窑瓷器数量最多、质量最精的国家级博物馆。

宣德款青花折枝花纹碗

万历款五彩人物故事图盘

　　《笔走龙蛇——傅山及明末清初名家行草书特展》展示张瑞图、黄道周、倪元璐、王铎、傅山等五位明末清初书法家近200件作品，包括11件太原市馆藏傅山书画作品。

　　张瑞图、黄道周、倪元璐、王铎、傅山是明末清初这一特殊历史时期最具个性、最富创新精神的书法家，他们在艺术上最重要的贡献就在于他们能够在董派柔媚书风笼罩下，以“敏而好古”为号召，力矫积习，独辟蹊径，另树一帜。他们的书风以硬倔、怪异见胜，为我国古老的书法艺术注入了雄强之势。

“傅山及明末清初名家行草书特展”展览现场

　　本展览分为两期，展品共计近两百件，是从故宫博物院所藏张瑞图、黄道周、倪元璐、王铎、傅山等5家300多件法书作品中遴选出的精品，以期有助于有关学者进行深入的研究。在明末清初书坛得到极大发展的是行草书，其中显著的特点之一就是大幅行草立轴的出现，张瑞图、黄道周、倪元璐、王铎、傅山均以擅写大幅著称于世，故此展览以立轴为主，按张、黄、倪、王、傅5家序列展出。各家作品中署有年款的，按先后序列，没有纪年的作品，尚能考出作品的大致创作时间，则依然按所考时间序列，不能确定时间的作品，统一序列在最后。

王铎 行书临王操之（王羲之六子）等帖扇面

　　《紫禁藏影——故宫博物院藏老照片展》聚焦清末至民国初期，设置“宫苑留影”“群像浮光”“兴业图强”“禁宫新生”四个单元，选出800余帧影像，再现当时的建筑风貌、人生百态、社会变迁和王朝更替。

　　相比于瓷器、珐琅工艺等需要一定的欣赏门槛的作品，这个展览中的老照片清晰呈现了我们所熟悉的人物的历史瞬间，引起游客较大的兴趣和讨论。

《紫禁藏影——故宫博物院藏老照片展》现场

　　《数字故宫?万象紫禁》数字展示厅，利用近年来故宫博物院数字化技术应用成果，通过专题片、互动体验等方式，从古建筑、历史事件、文物珍藏、传统技艺、宫廷生活等多个角度，力求全面生动地讲解故宫蕴含的丰富信息和背后的故事，观众将在传统文化与现代科技的交融中感受故宫独特的魅力。

　　《日丽紫禁——明清官式建筑展》分为“琉”光溢彩、移天缩地、金窗绣户、宫阙流芳四部分，集中展示紫禁城建筑的匾联、装修、琉璃瓦件和烫样。无论是技艺、还是用料，这些展品都从不同的角度展示出了明清时期宫殿建筑的华美与考究，同时，我们也希望借由这些承载着历史技艺的建筑构件，能够在观众心中搭建出明清盛世时辉煌灿烂的皇家宫殿。

卷草牡丹栏杆罩局部

匾额

　　单霁翔院长在开幕式上说：“让文物活起来，就需要让更多人走进博物馆，同时让文化走进人们的生活。故宫博物院藏品丰富且传承有绪。使院藏文物资源得到更好地利用，实现让文物活起来，始终是故宫博物院努力的方向，而举办展览则是达到这一目标的重要途径。故宫博物院的数项展览同时集中来到一个博物馆展出尚属首次，此次系列展览也是太原博物馆开馆后的首展，这一展览形式对双方来说都是一次全新的挑战，我们相信本次展览将带给观众全新的体验与感受。”

展览现场

　　今年3月，山西省人民政府与故宫博物院签订《合作框架协议》，计划在合办展览、古建保护、文物修复、藏品展陈、数字化展示、文博宣传、人才培养、学术交流、文创产品研发等方面开展全方位合作，促进文化遗产保护和博物馆事业不断发展和进步。根据协议，双方约定以借展方式在太原展出故宫博物院部分院藏珍贵文物。
