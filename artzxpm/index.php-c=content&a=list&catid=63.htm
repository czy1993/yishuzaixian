配送常见问题

1.运费有剩余如何处理?
2.超过送达时间，拍品仍未收到，该如何处理？
3.不同的竟买人，是否可以根据客户要求合并发货？
4.拍品出现破损，其邮费和保险费是否可以退款？
5.是否可以委托他人签收和到公司提货？
6.出现漏寄或错寄时，发生的补寄或寄回的邮费由谁承担？
7.如因客户提供的地址有误，造成的拍品退回，公司补寄的运费由谁来承担？
8.如拍品部分出现破损，是否可以拒收？
9.客户是否可以自己委托到公司提货？
10.对系统无法计算运费的拍品如何支付运费？
11.如客户不要求对拍品保险，该如何操作？
12.对破损拍品的理赔，能否全额赔偿？
13.书画破损后，该如何处理？
14.为何瓷器书画要分两个包裹发货？
（1）运费有剩余如何处理?
答：因公司与快递公司以月为单位结算，所以客户实际发生的邮费须在每月15日方能统计出来，我们会在月底对运费加以核对，如有运费剩余我们会通过邮件通知您剩余金额，如出现运费不足的情况，也会通知您，在下次汇款时补足。
（2）超过送达时间，拍品仍未收到，该如何处理？ 
答：如邮件在规定的时间内未送达，您可以通过网络留言、电子邮件或电话方式，告诉我们，以便我们及时为您办理邮件查询。
（3）不同的竟买人，是否可以根据客户要求合并发货？
答：可以一同发货，但需经双方共同确认，并需要您汇款时在备注栏中注明，内容包括：拍品流水号，买受人姓名，邮寄地址等信息。
（4）拍品出现破损，其邮费和保险费是否可以退款？
答：在物品出现破损时，邮费和保险费为实际发生的费用，都不能给予退款。
（5）是否可以委托他人签收和到公司提货？
答：可以。在委托他人签收或提货时，请务必让委托人带有效证件，以便登记便于查询。
（6）出现漏寄或错寄时，发生的补寄或寄回的邮费由谁承担？
答：因为此情况，为公司工作失误所致，故邮费由公司承担。
（7）如因客户提供的地址有误，造成的拍品退回，公司补寄的运费由谁来承担？
答：补寄运费由客户承担（因公司按客户提供地址发货）。
（8）如拍品部分出现破损，是否可以拒收？
答：如出现部分破损，请您不要拒收，我们会根据破损情况进行理赔，如须要把破损拍品寄回，我们会委托快递公司上门取回，费用由公司承担；因拒收后，寄回的途中，会造成拍品的二次破损。
（9）客户是否可以自己委托到公司提货？
答：可以。您需要提前把快递公司的名称告知我们，以便我们核对，必要时，我们会电话与您确认，以保证拍品的安全。
（10）对系统无法计算运费的拍品如何支付运费？
答：因拍品的种类繁多，对少数不规则拍品，系统还暂不能自动计算运费，您可以通过网络留言或邮件、电话与我们联系，我们会及时人工给您估算运费。
（11）如客户不要求对拍品保险，该如何操作？
答：可以，您需要通过汇款单备注栏明确注明。
（12）对破损拍品的理赔，能否全额赔偿？
答：对完全破损的拍品可以全额赔偿，但对部分破损的拍品，邮局视破损情况，给予最大限度的赔偿。
（13）书画破损后，该如何处理？
答：因书画的特殊性，书画破损通常在不缺肉的情况下，可以完全修复，您可以选择自己修复，费用由公司承担；或邮寄回我们公司，我们将免费修复后，再寄给您。
（14）为何瓷器书画要分两个包裹发货？
答：瓷器与字画的尺寸差别较大，一般情况下，我们会用单独两个包裹邮寄；如书画尺寸小于瓷器尺寸时，可以用一个包裹邮寄。
