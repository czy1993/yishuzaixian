龙泉青瓷走向世界 85件珍品亮相联合国总部

来源：admin     发布时间：2018-09-15

       现场嘉宾欣赏展品。龙泉宣传部 提供


　　中新网丽水9月11日电(记者 周禹龙 实习生 施紫楠)北京时间9月10日，“人类非遗·中华经典”——2018龙泉青瓷巡展·纽约站，在美国纽约曼哈顿区联合国总部大厦四楼举行，共展出81位龙泉青瓷艺人的85件青瓷珍品。

　　历史上，龙泉青瓷就是中国与世界交流的文化使者，在宋、元及明前期，参与开拓了漫长的海上丝绸之路，产品远销亚非欧三大洲50多个国家。

　　龙泉市人民政府市长吴松平介绍，龙泉青瓷在当代得到了很好传承与发展，近年来，其相继在G20杭州峰会、“一带一路”国际合作高峰论坛领导人圆桌峰会等国际大会上大放异彩。

　　此次活动以“海上丝绸之路”为切入点，分“起源”“流淌”“融合”三大篇章，重现“青瓷之路、开放之路、辉煌之路”。



      85件珍品亮相联合国总部。龙泉宣传部 提供

　　“起源篇”通过传统青瓷艺术品的展示，辅以手绘青瓷传统烧制工艺流程图版介绍，讲述青瓷千年历史；“流淌篇”以现代青瓷艺术品(以新锐中青年作品为主)和龙泉青瓷参与国际重要活动的图版形式，展现当代龙泉青瓷生生不息的活力；“融合篇”以日用茶具、餐具、文房、花器、香器、首饰类等青瓷工艺品为主，通过茶道表演、茶席展示及生活情景式展陈，体会青瓷千年魅力，拥抱东方品质生活。

　　工艺美术师支持的代表作《青衣》在此次活动中展出，主要取材于中国传统服饰，有着水墨画风。

　　“能将作品放在联合国级别的展会是我的荣幸。”在支持看来，依托国际化的平台，青瓷产业焕发出了更加蓬勃的生命力，为此也激励着他创造更好的作品。


       现场嘉宾欣赏展品。龙泉宣传部 提供

　　中国工艺美术大师胡兆雄表示，“人类非遗·中华经典”2018龙泉青瓷巡展是推动青瓷走向世界、走进千家万户的一次重要活动。“我们相信龙泉青瓷，一定能让世界各国的人们都喜爱、收藏、享用，开启中国龙泉青瓷迈向国际化全新路径。”

　　据了解，在联合国总部展结束后，龙泉还将在纽约SOHO区举办为期一周的展销，并组织企业团开展青瓷产品推介，与美国有关企业签订青瓷销售合作协议，既宣传“人类非遗”文化，更让龙泉青瓷文创产品走向国际市场。(完)
