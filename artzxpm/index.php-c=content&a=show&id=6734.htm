印度被盗走的天价金饭盒找到了 小偷竟然用来吃饭

来源：admin     发布时间：2018-09-15

　印度警方9月11日称已经抓获了两名盗窃印度王公金饰品的窃贼，此前失窃的印度王公使用的金饭盒已经被找回，丢失的红宝石和金茶杯，金茶碟和金茶匙也一同被找到。而据印度警方称，与其他盗窃金饰品的窃贼一心只想转手卖掉不同，这两个小偷还用偷来的金饭盒美美的吃一顿饭。

印度警方展示此前被盗的金饭盒

　　这批价值700万美元（540万英镑）的物品曾经属于印度海德拉巴的最后一位尼扎姆（国王）米尔奥斯曼？阿里汗（Mir Osman Ali Khan），印度媒体称他曾经是世界上最富有的人。据报道这批宝物此前陈列在海德拉巴王国的王宫改建的博物馆内，该博物馆2000年向公众开放。9月3日一伙盗贼闯进博物馆偷走了包括黄金打造的镶钻饭盒、红宝石、金茶杯、金茶碟和金茶匙，这些物品重达3公斤。

此次失窃的金饭盒

　　据印度警方的消息称，这两名窃贼来自印度的南部城市海德拉巴。在9月3日盗窃得手后这两名男子逃往西部城市孟买，希望能在那里找到这批赃物的买主，他们在当地一家五星级酒店度过了几天的“生活”。然而，由于印度警方展开了大规模搜捕，他们无法找到买主因此返回了海德拉巴。由于事发时小偷破坏了馆内的32个摄像头，所以未录下任何影像。幸好在宫殿附近一辆摩托车上的摄像头发现了小偷驾驶的摩托车特征，随后顺藤摸瓜最终找到了这两个小偷。

印度国菜“咖喱”

　　根据报道，警方发现他们躲在孟买一家豪华酒店时候，他们曾用金饭盒来吃着他们的午餐，享受了一把印度王公的生活。不过很显然这顿饭的代价有些大，不知道在狱中他们还会怀念在五星级酒店用金饭盒吃饭的美好下午吗？
