【艺术在线】收藏之路，让我们走得更好

来源：admin     发布时间：2018-09-12

　　讲真，作为一名收藏爱好者，我花费了十年的时间，才搞明白要想在收藏界捡漏，这简直是不啻于奇迹发生。十年来，关于收藏的心酸血泪，我毫不夸张地说可以写一本书。之前参加过很多拍卖会，良莠不齐，时而上当，耗费了钱不说，有时还买回一些赝品，遭到了家人和朋友的嘲笑。艺术品收藏，让我一度心灰意冷，甚至想放弃这个爱好。但出于对艺术品的热爱，我痛定思痛，一方面深入研究收藏知识，另一方面会选择可靠性高、专业度强的拍卖公司。

　　最近，藏友圈中听说有一场秋季拍卖会即将于9月23日在无锡美丽都大酒店举行。8月10日-9月10日期间，这场秋季拍卖会将面向个人、藏品机构全面征集拍卖艺术品，范围包括：中国书画(包括古代、近现代、当代)，现当代艺术(油画、雕塑、影像等)，古董珍玩(瓷器、玉器、紫砂、家具、工艺品等)，古籍，珠宝钟表尚品，邮品钱币等，可谓是品种繁多。这次的拍卖会由金远见(北京)国际拍卖有限公司、北京中塑文化艺术有限公司江苏分公司和北京远航方舟教育文化交流有限公司无锡艺术品分公司共同举办。

　　我还打听到本次秋季拍卖会上，所有征集藏品将进行线上图片鉴定和线下专家联合实物鉴审，做到保真、保值、保质，从而保证拍卖品真实性和可靠性。

　　我特意在网上搜集了主办方的信息：北京中塑文化艺术有限公司旗下拥有“北京远航方舟教育文化交流有限公司”“金远见(北京)国际拍卖有限公司”“北京中投精微防伪有限公司”“中投昊天基金管理(北京)有限公司”等七家企业。同时，它也是文化部艺术发展中心中国民间艺术品收藏评估委员会(简称“中艺评委”)授权的运营公司，可以代表中艺评委全权代理有关业务。北京中塑文化艺术有限公司还与文化部艺术发展中心中国民间艺术品收藏评估委员会签署协议，正式打造一体化合作运营体系。

　　由此可见，本次拍卖会的主办方是有资质、有权威性的。而且他们在去年11月成功举办过一次拍卖会，据说当天成交203件，成交金额达到287万元。

　　我不禁期待地搓搓手，想把我收藏的那些“宝贝”们，可以出来晒晒太阳啦。我已经迫不及待地在心里盘算着要拿哪几件藏品去参加这次征集和拍卖。在这场拍卖会上，或许能收获几件和我有缘的艺术品。希望通过这次权威性的拍卖会，我能和其他收藏爱好者一起交流学习收藏知识和经验，提高鉴赏能力，让我们的艺术品收藏之路走得更好。
