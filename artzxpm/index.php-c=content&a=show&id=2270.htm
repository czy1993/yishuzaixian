【艺术在线】2017年会是现当代艺术市场的“拐点”吗？

来源：admin     发布时间：2018-01-25

早在2017年拍卖季开始之前，便有市场专家预测今年将是中国现当代艺术市场触底反弹的一年，毕竟经过漫长的调整期，曾经的泡沫大部分已被挤掉，甚至有些矫枉过正。而这一年里，香港和内地市场此起彼伏的高价接力，以及曾经各行其是的现当代艺术市场价值观的趋同，则用大量真金白银为“拐点说”增加了厚重砝码。

2017年，现当代艺术二级市场出现许多重要作品，诸多藏家用真金白银缔造了一个又一个高价，而这似乎预示着新一轮市场行情的来临

图表1-2011-2017年10家指标拍卖行现当代艺术板块行情整体走势图

　　(样本拍卖行：香港蘇富比、佳士得、北京保利、中国嘉德、 北京匡时、保利香港、西泠拍卖、翰海拍卖、诚轩、华艺国际)

　　通过分析内地和香港地区10家指标拍卖公司的成交数据发现，2017年现当代艺术板块的成交总额为51.04亿元，同比2016年增长了5.69%。其中2017年秋季表现尤其出色，包括香港苏富比、佳士得以及中国嘉德在内的多家拍卖公司都在该季取得了近几年以来的最高单季成交。2017年秋拍10家拍卖公司共取得29.25亿元的成交额，为近十年来仅次于2013年秋季的第二峰值。两地市场用诸多高价和更有针对性的经营策略，勾勒出一条明显的复苏曲线。

　　然而在大藏家、大拍卖行高歌猛进的同时，作为市场基础的中低端市场、以及新兴板块却也面临着严峻的危机，在冰与火的交织下，2017年真会如愿成为现当代艺术市场的“拐点”吗?

　　价值回归，早期油画能扛起市场的大旗吗？

图表2-2017年度现当代艺术高价TOP，榜单统计了2017年超3000万元成交的23件拍品，标红为刷新个人拍卖纪录的成交

　　作为市场复苏的号角，高价拍品无疑是最受行业内外关注的焦点。然而与往年不同的是，在今年的现当代艺术高价榜中，20世纪早期油画占据了绝大多数的席位，显示出超越当代艺术成为市场主角的潜质。如上图所示，在2017年23件超3000万元成交的现当代艺术拍品中，20世纪艺术占19席;而在64件超千万成交的拍品当中，47件为20世纪艺术。

赵无极《29.01.64》2017年香港佳士得 成交价：2.026亿港元（折合1.724亿元）

　　在高价拍品的构成方面，赵无极无疑是2017年现当代艺术市场风头最盛的艺术家。其1964年作品《29.09.64》和《29.01.64》不仅在佳士得香港的两季拍卖中以1.356亿元和1.724亿元两度刷新个人记录，而且全年共有18件作品超千万元成交，8.56亿元的年度成交总额，也创造了个人历史新高。

常玉《白瓶花卉》115x88cm 布面油彩 1931年 成交价：7446万港元（折合6604.6万元）

　　同样作为20世纪市场票房担当的常玉和吴冠中，今年各有5件和3件作品超3000万元成交。其中受益于台湾历史博物馆“相思巴黎—常玉的艺术”大展的举办，常玉的油画在高端市场中得到了非常热烈的反响，3.58亿元的年度成交额同样是常玉历史新高，反映出藏家对20世纪艺术大师作品稳定的购藏信心。

朱沅芷《工业之轮在纽约》1932年 成交价：1.0528亿港元（折合8387.86万元）

　　值得一提的是，第一代旅美华裔画家朱沅芷也在今年取得了大幅度的价格突破，其艺术生涯最具代表性的作品《工业之轮在纽约》，于香港蘇富比2017年秋拍中以8738.86万元成交，创造了朱沅芷个人拍卖最高价纪录。

相较于香港市场依托20世纪艺术大师们的持续昂扬，大陆地区该板块在今年发生的变化则表现得更为深刻。2017年秋季，中国嘉德在大陆地区首次呈现了中国二十世纪油画夜场，这场标杆性的夜场拍卖所选择的艺术家名单囊括了吴冠中、苏天赐、吴大羽、丁衍庸、萧淑芳等老一辈油画艺术家，多位艺术家在这场拍卖中创造了其个人拍卖的最高价纪录，热烈的反响正是近几年过内早期油画收藏趋势的集中体现。

苏天赐《苏南初春》80x64cm 布面油彩 1980年代 成交价：805万元

　　此外，在中国早期油画重要发生地的杭州和广州，西泠拍卖和华艺国际也在2017年秋季分别组织了“从国立艺专到中国美院——‘国’‘美’九十年纪念专场”和“广东美术百年作品”专场，通过地方性美术史线索的梳理，串联起众多早期油画家及作品。其中，吴大羽罕见的50年代转折期作品《瓶花》在西泠拍卖中以4140万元成交，大幅刷新了其个人拍卖纪录;而在华艺国际拍卖中，也有中国西画第一人李铁夫的新纪录诞生。

吴大羽《瓶花》60.3x50.3cm 布面油彩 1950年代 成交价：4140万元

　　20世纪早期油画从2005年中国艺术市场兴起后在内地就一直广受关注，但因为市场流通量小、伪作问题以及艺术价值未被广泛认可等多方面原因，市场一直未有太大的起色。然而在近几年中国当代艺术进入调整期的微妙时段里，艺术价值更加确定，只待有心人发掘的20世纪早期油画开始异军突起。

　　对此，资深艺术品经纪人伍劲认为，“当代艺术起飞以后，‘二十世纪’就被忽略了，所以当代艺术停顿以后反而给它一个被关注的机会。但‘二十世纪’的被重视也受惠于当代艺术的成长，就是这个饼大了，然后可以分一部分给二十世纪的油画。”

图表-3 7家内地指标拍卖行在中国二十世纪艺术板块的成交走势图

　　从近5年内地市场20世纪早期油画的走势来看，自2014年随市场整体进入阶段性的低谷以后，早期油画在内地市场的成交额和市场份额一直呈快速上升的趋势。加之其基础价格较低、风险性也相对较小的特性，越来越多拍卖行加大了对该板块的投入。除前文所述的嘉德、西泠和华艺外，北京匡时近几年每季大拍中也都有早期油画的专场呈现，保利拍卖系依托吴冠中等早期艺术家的征集资源优势，屡有优异表现。

　　随着板块规模的逐渐提升，拍卖公司征集要求越来越高，市场也自发地开始了对被忽视的早期油画家的挖掘。如在2017年创造新纪录的吴大羽，便是2015年至今最受关注的市场焦点。此外，在2015年数家拍卖行集中举行过多位早期油画家的个案专场，包括沙耆、王济远、张光宇、李宗津、王肇民等今天市场中耳熟能详的艺术家，都是在这一阶段中逐渐涌现。

图表4-2012-2017年吴大羽成交走势图

　　而当早期油画家个案的越来越多，由点到线，再由线到面的梳理趋势也从2017年开始变得愈发明显。势象空间创始人李大钧认为，“客观上讲，从90年代初早期油画家的价值就不断被认知，只是前期话语权多掌握在台湾、香港市场手中……在很长一段时期里，内地对20世纪现代艺术价值的认知已经基本上归零了，现在被重新发掘也是一个自然的过程。西方艺术市场从来都是一环一环地递进，从印象派到现代艺术到战后艺术层层推进，价格是非常合理的，它没有哪个艺术史的环节被忽视，我们缺的正是这个。”

李铁夫《刘思健肖像》92.5x64cm 布面油画 1943年 成交价：1081万元

　　虽然就板块深度而言， 20世纪早期油画板块目前还没有海外“印象派及现代艺术”那样的实力，但其潜力有目共睹，在以后的市场中也很可能将是和当代艺术并驾齐驱的重要板块。不过，由于目前中国学术机构的乏力，如果将美术史构建的话语权过多让渡到市场手中，也难免会被所谓的“潮流”带起不理性，长远来看不确定风险也更高。在资深艺术品经纪人、市场专家李苏桥看来，“20世纪油画的历史梳理其实刚开始，艺术家们的历史地位相对应的市场地位，影响美术史的艺术品推定，这些艺术市场最擅长的事情，很多事都还没有分晓。”

哪些当代艺术家引领了市场的“触底反弹”？

　　自中国当代艺术市场进入调整期以来，2008年前后的所谓“当代艺术”被重新定义，其间流行过的板块如“卡通”、“政治波普”、“新水墨”、“抽象”等大行情都已不在，市场格局依旧在不断洗牌。但也有部分艺术家能够历经调整而屹立不倒，并在新的市场环境中，构建起新的当代艺术市场格局。

图表-5 2017年当代艺术高价TOP10

　　通过统计2017年度当代艺术高价TOP10，我们发现了一些熟悉面孔重回榜单。如经典写实油画的代表人物陈逸飞，历经长期的市场沉寂，在2017年嘉德秋拍中，其1993年经典作品《玉堂春暖》以1.495亿元的被龙美术馆收藏，再度震撼了市场。主张中西融合，并在绘画技法上独树一帜的周春芽，不仅在2017年有两件作品进入榜单，其热络的买气更被部分市场专家解读为具有东方精神的当代艺术成为市场新线索的前兆。此外进榜的曾梵志、张晓刚、罗中立和刘炜等艺术家都是近几年当代艺术市场的中坚人物。

陈逸飞《玉堂春暖》在嘉德秋拍中1.495亿元被龙美术馆竞得，不仅刷新了陈逸飞个人拍卖纪录，也是2017年内地油画拍卖的最高价

　　2017年当代艺术顶端市场极为活跃，各拍卖行的重点拍品大多得到了新老藏家的积极响应，也因此诞生了许多令人意想不到的高价。中国嘉德二十世纪及当代艺术部总经理李艳锋将这股高端市场的竞买热情总结为“买家对优秀作品的明确共识”。

曾梵志1996年的作品《面具系列1996 No.6》在保利香港2017春拍中以10502万港元(9357.2万元)售出

　　而从高价作品的成交地点分布来看，8件作品成交于内地拍卖行的事实，似乎昭示着过去一直由港台主导的中国油画市场已被撼动，中国油画的主流价值观开始转向内地。大陆拍卖行在经历了当代艺术市场持续多年的调整行情里，开始掌握了弱市中的经营之道：一方面着力于成熟艺术家的美术史线索梳理，另一方面将更多精力集中在艺术家早期作品或重要作品的挖掘中，彰显二级市场的资源优势和特点。

图表-6 2013年——2017年春北京保利、中国嘉德和北京匡时三家拍卖行现当代艺术板块成交量和成交额走势图(数据来源\制图：雅昌艺术网)

　　对于北京三家拍卖行的现当代艺术部分而言，今年的行情走势可谓是跌宕起伏。在市场环境并未有显著变化，成交量也基本持平的前提下，三家拍卖在春秋两季却分别取得了近5年最差和最好的迥异成绩。显而易见，两季拍卖最大的差异主要集中在于高价市场的表现。

　　中国嘉德在2017秋季这场高价争夺战中扮演最重要的角色，在陈逸飞《玉堂春暖》1.495亿元成交和周春芽《中国风景》4427.5万元成交的强势带动下，中国嘉德现当代艺术部在今年秋拍共取得了4.5亿元的历史最好成绩。北京保利则是在张晓刚、周春芽、常玉及曾梵志等艺术家的高价作品引领下，较春拍总成交额提升了近20%。

图表-7 2017年中国当代艺术家成交总额TOP30

　　2017年度的当代艺术家成交总额TOP30方面，曾梵志以微弱优势，力压周春芽位居榜首，而这也是他连续第五年(2013-2017)蝉联榜首，显示了坚实的市场地位。榜单中同样排名靠前的周春芽(历年排名为2-2-2-7-2)、张晓刚(排名为10-3-3-4-4)和刘炜(排名为7-4-5-2-5)则均是近五年来第4度进入榜单前5。这四位艺术家经历了2008年以来的数度行情起伏，依然在香港和内地市场中保持着旺盛的市场活力，堪新一代中坚力量。

周春芽1993年《中国风景》在中国嘉德2017年秋拍中以4427.5万元成交，刷新了其个人拍卖纪录

　　此外，2017年表现明显回暖的艺术家还包括：罗中立(4-6-14-未上榜-5)、刘野(5-14-19-24-7)和王广义(未上榜-21-27-21-8)。新进入榜单的萧勤在香港蘇富比两度刷新纪录，积淀多年后完成了从量到质的飞跃。

罗中立1982年创作的第三幅《春蚕》在中国嘉德春拍中以4945万元被泰康系竞得，刷新了罗中立的最高拍卖纪录

　　写实油画家在今年表现回勇，陈逸飞、罗中立、艾轩、杨飞云、徐芒耀、冷军、庞茂琨和陈丹青等8位艺术家都进入成交总榜。对于写实油画在今年的回暖表现，中国嘉德油画部高级业务经理孟禄新表示：“在调整行情中，经典写实绘画依然保持着非常深厚的受众基本，而其中的重要作品更是吸引着大量重要买家的关注，这也是经典油画的魅力所在。”

　　艺·凯旋画廊负责人、资深艺术市场专家李兰芳认为，2017年秋拍是从2011年以来内地市场最振奋人心的一个季度，她表示：“一直以为2018年市场会好起来，但没想到在2017年的冬季提前来到。之所以有这样的爆发行情，一方面是由于市场中涌现了许多好作品，另一方面不论资本市场还是藏家的审美取向都统一到东方美学价值上，这也为市场后续发展带来了更大的潜力。”

青年艺术，走出新路还是重蹈覆辙？

图表-8 2008-2017年70后艺术家市场成交额及成交量走势图 

　　近五年来，出生于70、80年代的艺术家开始在拍卖会中崭露头角。这些年轻的艺术家为最具威信的画廊所支持，其作品的拍卖价格均达到五年前难以想象的价位。但经过一价格快速上涨后，今年的青年艺术板块显出疲态，曾经集体出场的 70、80 后艺术家行情在一段时间的沉淀后开始分化。

图表-9 2017年，70后艺术家成交TOP10

　　从2017年70后青年艺术家单价TOP10来看，今年青年艺术市场的市场名单与往年差别不大，像刘韡、郝量、贾蔼力、仇晓飞和陈飞均是从2013年起引领青年艺术热潮的热点人物。然而从更宏观的层面来看，2017年，甚至近几年的青年艺术市场在经历了短暂的高峰期后，已经进入下行调整通道，甚至从今年青年艺术春秋两季的对比来看，秋季成交额(3258万元)只有春季(6172万元)的近一半。而且在全年的成交额、上拍量、成交率等多项数据方面均较去年出现滑落。

刘韡《真实的维度No.18》布面油画 300×150cm;300×200cm;300×150cm 2015年作 成交价：644万元

　　北京保利是今年对青年艺术家作品推荐力度最大的拍卖行，不仅凭借“尤伦斯收藏专场”在榜单中独占5席，还在青年艺术年度成交额和成交量方面分占总量的57.11%和39.34%。这一方面反映了北京保利今年在青年艺术板块的成绩，另一方面也暗示了其他拍卖行对该板块投入的减少。

　　在今年的榜单中，一些前几年在市场中炙手可热的人物跌出榜单，但也有一些新人进榜。这在资深艺术品经纪人李苏桥看来并不值得奇怪：“优胜劣汰是市场的基本原则，这在70、80后这一批艺术家的市场上反映得更加强烈，全世界范围内都是这样子。”

郝量2009年的作品《移用解剖学系列》在北京匡时春拍中以575万元成交，是2017年40岁以下年轻艺术家作品中的最高单件成交

　　青年艺术家在市场中的快速起伏，与年轻人身上所伴随的不确定性相关。这种不确定性一方面源于艺术家持续创作过程中的未知变数;就像一位年轻藏家讲到的：“当接下来几年没有看到艺术家的进步的时候，再买作品的时候就会开始迟疑，而更可怕的是，更多时候看到的是艺术家的退步。”另一方面则来自市场，其中包含了艺术家与市场的合作程度以及一级市场和二级市场的平衡关系等博弈，需要每个个体自己把握。

贾蔼力《无题》布面油画 220×300cm 2007年作 2017年北京荣宝秋拍成交价：448.5万元
