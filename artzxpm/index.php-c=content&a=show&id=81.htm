崔如琢巨作《听声》亮相保利香港五周年秋拍

来源：admin     发布时间：2017-09-11

崔如琢作为全球最贵在世艺术家，仍不断在中国传统艺术道路上前进，正如李瑞环对其评价：“继往开来一大家。”继往是说对历史和传统的态度，不仅是笔墨技法层面上的传承，同时包括对中国文化的自觉与自信，开来则是其缘此而建树的不凡成就。崔如琢在中国传统文化和现代文化之间找到新的接榫点，使自己的艺术在新的历史条件下再放异彩。作为一个当代中国艺术家，他是一个肩负历史使命的行进者，为中国艺术与文化事业做出了不可估量的贡献。其作品气象万千，精湛的艺术手法和苍茫厚的笔墨气势，打开了一个气韵生动的境界。

　　此次保利香港2017秋拍，将继续呈现“太璞如琢”—崔如琢精品专场[Ⅸ]，包含积墨与指墨多种艺术手法的精彩佳作，特别呈现达六十八平尺的鸿篇巨作《听声》。此件作品为估价待询，本专场其余拍品总标底达港币3.55亿港元。

　　Lot 1211 崔如琢《听声》 2013年作 设色纸本 镜心 143 x 524 cm 估价待询

　　《听声》画面中积墨画法绘制的墨叶与姿态各异的荷花相互掩映。画面一侧，磐石矗立，与摇曳的荷叶形成一刚一柔的对比。崔如琢之积墨，极具己法，无论是混沌还是清晰处，都施展自如，使画面在厚重之中又见细节与层次，在无中见有，蕴含着哲思妙义。作品以《听声》为名，捕捉在充盈饱满的视觉感受之外的多重感官，如临其境。

　　Lot 1205 崔如琢《日出江花红胜火，春来江水绿如蓝》 2013年作

　　设色纸本 镜心 143 x 289.5 cm 估价：HK$ 36,000,000 - 42,000,000

　　此件作品描绘春雨朦胧中红花遍染，山峦下细舟靠岸，远山与近景相互呼应，又有楼阁点缀其中，使画面更加鲜活而富诗意。

　　Lot 1207 崔如琢《冷碧新秋水，淡香醉万家》 2016年作

　　设色纸本 镜心 547.5 x 66 cm 估价：HK$ 32,000,000 - 37,000,000

　　此件五米余长的手卷作品，色彩清新淡雅，呈现残荷碧水之景，颇的潘天寿写意花卉之意，又在其精髓之上融入崔如琢的艺术创造力。

　　Lot 1209 崔如琢《野水白云外》 2011年作 设色纸本 镜心

　　143 x 365 cm 估价：HK$ 48,000,000 - 55,000,00

　　大尺幅作品《野水白云外》呈现雄浑山峦矗立于云海之中，积墨手法运用于山峰的描绘之中，淡云与远树的淡墨渲染，山石的重墨叠加，落墨处的浑黑精明，体现着其张弛有度，收放自如的境界。

　　Lot 1210 崔如琢《十六开册页》 2011-2012年作 设色纸本 镜心

　　书法: 45×35cm. 册页: 48×38.5cm.×16 估价：HK$ 28,000,000 - 35,000,000

　　此套崔如琢指墨册页作品包括一件指墨书法及十六件指墨山水册页，内容丰富，描绘四时不同之景。崔如琢高超的指墨技法给予了作品灵动与生机，春芽、夏雨、秋风、冬雪都在画面之中无限展开。

　　保利香港2017五周年秋拍

　　预展及拍卖

　　时间：9月30日-10月3日

　　地点：香港君悦酒店
