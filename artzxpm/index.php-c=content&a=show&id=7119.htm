       很多藏友一开始都会问我很多问题，我这边把大家常问的几个问题和大家分享下：

       第一、你们这边收购藏品吗？

       不管"收购"这词是哪家拍卖公司或者是谁告诉你们的，这是业界最典型收取高昂鉴定费的一个骗术。作为正规拍卖公司收购是国家明令禁止的，这是违反法律的事情、希望不要去做，也引起重视！

       第二、你们先帮我找找买家？

       公司都不是做慈善的，要是说去找免费的、这肯定是不现实的，目前这个市场上藏品是买家的几十倍！当今市场找买家比征集藏品难太多太多，所以没有人会在意你的藏品！如果有业务员天天夸你东西好，追着你死缠烂打，叫你跟他们合作，说已有买家看中要买，或者说免费给你卖。你要当心了那这家公司有可能是骗子公司！

       第三、你们这里有没有服务费用、是免费的吗？

       只要是正规公司肯定都有服务费用，并不是不收钱的就是正规的、收钱的就是骗人的，但藏家要懂得区分。很多人可能就会考虑到这么麻烦、还要花钱，不会去做、想找简单、快捷的渠道，其实不然、如果要卖这是必须要做的，不要想着占便宜、走捷径，正是因为很多人不会考虑到这些才往往着了骗子的道。

       如今，随着古玩市场的不断升温，越来越多的人将资金投向古董珍玩和名人字画等收藏品，以期获得不菲的收益。古玩市场上充斥着不少赝品，盲目投资会付出巨大代价。因此，投资者必须了解古玩投资的几点要诀：

       一、藏家希望购买的藏品要少而精，且有升值空间。如果您是新手，不妨选择一种长期稳定的藏品或从小件精品入手。

       二、真正的藏家都是“只要物有所值，肯花一定代价购买上乘古玩”的心理。只要是精品，就可大胆买入，否则会错过赢利机会。

       三、藏家希望自己的收藏是“宁要瓦全，不要玉碎”，所以请妥善保管藏品，使其保持最佳状态。

       其实，古玩古玩，本身就是一件很好玩的事情，收藏之乐存乎于心，心动则行动，不管价格的高低，只凭内心的悸动。而古玩古董的交易却不是一个能一味求快的过程，否则心急吃不了热豆腐，只能伤害到自己。那些不愿意等，不能等，抱着一眼就成交，一夜暴富梦的投机者，永远也不会成功，因为这不是玩的真谛。

       说了这么多，其实古玩交易，这个从古到今都没能理清的事情，我觉得任何买卖是靠买卖双方共同来完成的，在买卖时，今天你是买家，你要站在卖家的角度看问题，这么便宜卖家会卖吗？藏品符合吗？明天你是卖家时，你要站在买家角度看问题，人家花这么多钱，我藏品符合不？我提供的藏品服务买家以后出手方便不？只有这样你才能买到好的藏品，合理价格卖掉手上藏品。
