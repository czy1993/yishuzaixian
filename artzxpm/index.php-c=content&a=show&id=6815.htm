前泽友作的太空之旅会邀请谁 村上隆or草间弥生

来源：admin     发布时间：2018-09-19

　　要说科技与艺术相结合真的是已经司空见惯了，但今天这件事真的是不可错过，科技圈和收藏圈两大ICON一起造了个头条，前泽友作成了埃隆马斯克的首个实现绕月飞行的私人乘客，还在社交网络上发文说，将带多名艺术家一同前往太空。


　　据美联社18日报道，美国科技狂人埃隆·马斯克的太空运输公司SpaceX于当地时间周一晚上9时45分宣布，日本亿万富翁企业家前泽友作成为首位乘坐“大猎鹰”火箭（BFR）飞船绕月飞行的私人乘客。

　　那么，前泽友作会邀请哪位艺术家一起飞向月球呢？有微博网友猜测是带村上隆、草间弥生亦或是荒木经惟，你觉得呢？他在推特说：“如果毕加索近距离看到月球的话，他会画出什么样的画作呢？如果（披头士乐队成员）约翰·列侬围绕月球一圈，他会做出什么样的作曲呢？”

草间弥生作品

村上隆作品局部

　　他说：“我们每个人都有想象力和创造力”，他期待着艺术家们在看到从未见过的景色时所浮现的创造力和想象力。

　　据一位接近前泽友作的日本文艺界人士透露，前泽有作会挑选他发起设立的东京当代艺术基金会的合伙人一同前往。

　　这次环月旅行将花费至少数千万美元

　　据《纽约时报》报道，这次昂贵的SpaceX私人环月旅行将花费至少数千万美元。但今天的发布会上，马斯克和前泽友作均未透露价格。

　　而作为首位私人环月旅行者，前泽友作称，自己从小就喜欢月亮，近距离接触月球是其毕生的梦想，因此买下了整艘飞船的有限的所有座位，但鉴于独自一人旅行太孤独，他希望和很多人一起分享环月旅行。

　　前泽友作也表示，他希望在这次旅行中带上六到八位艺术家，这是他称之为Dear Moon的项目的一部分，并补充称它将于2023年推出。

　　前泽友作开了挂的人生从去年的一次拍卖开始

　　2017年在5月18日，纽约苏富比（微博）当代艺术晚间拍卖会上，他为艺术家Jean-Michel Basquiat 1982年的作品“无题” 支付了1.105亿美元。根据苏富比(微博)拍卖行的报道，1.105亿美元的拍卖价是1980年以后创作的任何艺术作品和美国艺术家创作的任何作品的最高拍卖价。前泽友作竞拍当晚，有五部Basquiat作品参与拍卖，其他四部作品售价不到2000万美元。与此同时，拍卖中的48件作品共售出3.192 亿美元，这意味着前泽友作个人的购买额，占当晚总和的三分之一以上。

前泽友作购买的 让-米谢·巴斯奎亚的油画《无题》 （1982）

前泽友作藏品 杰夫-昆斯《龙虾》（Lobster）

让-米切尔-巴斯奎特 《无题》 1982年作

　　此次收购同时打破了他自己创下的大部分Basquiat画作的收入纪录。就在他买下这幅画作的一年前，他以5730万美元的价格拍下了同一位艺术家的作品。在2016年的拍卖会上，前泽友作还买下了Bruce Nauman，Alexander Calder，Richard Prince和杰夫昆斯的作品，两天共花费9800万美元。

前泽友作

　　前泽友作人物介绍

　　前泽友作1975年生于日本千叶县镰谷市，是一名日本实业家，创立了Start Today公司以及日本电子商务巨头Zozotown。据福布斯估计，截至2017年3月，前泽有作的总资产为3330亿日元（约合人民币204亿元），位列世界富豪榜第630位，日本富豪榜第14位。

　　前泽友作也是有名的收藏家。他十分喜爱红酒，在自家的仓库里保存了数千瓶红酒，家中也收藏了多辆高级轿车。他还常常出现在当代艺术家拍卖会上，并且出手阔绰。
