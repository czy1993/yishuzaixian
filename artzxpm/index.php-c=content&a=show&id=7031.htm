无锡博物院10件国宝级文物首次亮相

来源：admin     发布时间：2018-09-29

　　今天，无锡博物院开放十周年暨《梁溪折桂——无锡博物院开放十周年特展》开幕。其中，作为十周年庆祝的“开幕大戏”，此次特展备受关注，因为，展览将首次集中展示无锡博物院馆藏十大“国宝级”文物，“以往，因为保存及各方面原因，这些宝贝很少示人，更不要说像这次这样集中展示了，机会十分难得。”无锡博物院相关负责人坦言，下次再集中展示的时间暂不可期，所以广大市民及文博爱好者们一定要抓住“机会”。

　　据了解，此次特展分为兵戈相见、应物象形、金奢银华、人间瓷话、紫玉金砂、泥塑大千、玉成其美、文心匠韵、砚田墨海、落纸云烟、石渠之宝、吴地风雅、梁溪画史等13个单元，汇集青铜器、玉器、瓷器、造像、泥塑、紫砂、金银器、竹木牙雕、文房及书画艺术等众多门类的精品，也是无锡博物院成立以来，首次集中“晒家底”。其中，刚刚通过“十大宝藏评选活动”，由广大市民票选出来的“十大宝藏”，最为亮眼，成为当之无愧的“主角”，堪称无锡博物院馆藏“精品中的精品”。

　　据悉，“十大宝藏”为：

　　1、春秋-吴王僚剑

2、元-倪瓒苔痕树影图轴

3、明-文徵明蕉石鸣琴图轴

4、南宋-龙泉窑梅子青釉鬲式炉

5、元-春水玉带扣

6、明-朱元璋手谕卷

7、五代-杨凝式韭花帖卷

8、清-光绪周阿生塑陈杏芳彩蟠桃会泥塑

9、清-杨芝山款西园雅集图核雕

10、元-“邓万四郎”款连生贵子纹高足金杯

　　每一件都蕴含着深厚的文化价值，代表着一段独特魅力的历史。比如说，《明朱元璋手谕卷》，讲的是朱元璋告知大将徐达，以后俘获敌军无需送来，可以就地正法，就显示出朱元璋在群雄纷争时代的杀伐决断和残忍气质。而《五代杨凝式韭花帖卷》，系清宫流出的秘藏珍本，后被无锡博物院收藏，从无锡老的博物馆开始至今这么多年，仅仅对外展出过3次，且每次展期很短，往往是在公众面前的“一闪而过”。

　　据介绍，2008年10月1日，由原无锡革命陈列馆、无锡博物馆和无锡科技馆“三馆合一”组建而成的无锡博物院正式对外开放，至今共接待观众630余万人次。籍着此次十周年特展，无锡博物院“智慧导览”功能也正式上线，安装APP后，参观者走到什么展品前，系统就会自动进行展品解说，相当具有服务性。

　　记者了解到，展览将至2019年1月8日。趁着这个国庆节，不妨到无锡博物院欣赏一番吧。
