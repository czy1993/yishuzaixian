【艺术在线】段俊平书法作品连续在各大拍卖会高价成交，书法审美回归于传统

来源：admin     发布时间：2018-09-13

      日前，段俊平先生的书作连连在各大拍卖会竞拍出高价，不但创造了各场书法作品每平尺价格新高，更释放出书画界近日的审美回归于传统的讯号。

秉现代情思而全古意风韵，间架和谐兼具自然美丽与古风雅趣，让书法回归于直觉审美和正大气象的传统美学，从形体到神韵，书法艺术之美于段俊平先生作品中体现得淋漓尽致。


      这种书法艺术之美，是2017荣宝秋拍中《寻隐者不遇》的隐逸风雅之美，段俊平先生初行笔闲适如行于山间，问答于童子，将魏晋放旷散发云游的意蕴，和唐代自由超逸的神气收于笔尖，信手挥洒肆意而不纷乱，颇有怀古沉吟又释然挥袖的往复味道，笔画间萧散疏淡之感如奇崛山石枯枝兀然而出，孤傲意气几欲破纸。

      这种书法艺术之美，是2018保利春拍中《沈周题画诗》的清润潇洒之美，段俊平先生此作笔意连绵直贯而下，大有行云流水，一路抛珠撒玉流淌过深山幽涧，激流交迸，凭空生出袅袅云烟之感。段俊平草书作品向来别具一格，笔法古朴自然无拘无束，有乱石铺路之概，在点画之间绽放独特的美感，飘逸中似见如画般的山水美景。

      这种书法艺术之美，是2018瀚海春拍中《福田花雨》的富贵堂皇之美，段俊平先生擅写诗词、山水、绝句，此类笔墨丰盈、具皇家气象的作品较为稀少，更显得格外珍贵。书作运笔浑厚圆转，笔意安闲优雅，开合有度，布局雍容端和，结构笔画饱满丰润，方圆兼备，体态悠闲，少见锋芒，温润自然，大有“贵而不自知”之感。

书法之美，无需评判，更不必以尺寸丈量，它就像清晨的露珠在花瓣上滚动着映照出第一道阳光一样璀璨美妙，像通体通红的锦鲤悠然穿梭于荷池甩开如长裙般的鱼尾一样优雅动人。

      这是笔墨阴阳调和书就的，穿梭千年时光依旧灵动的美丽，摆脱了“为艺术而艺术”的审丑误区，活得更加率性真挚的人们，开始追逐直击心灵、令人一瞬间体悟到感动和震撼的美。

      那是文人们初次执笔，屏息静观乌黑墨迹在雪白宣纸上游走，造出的鲜明又和谐的美丽。是书以咏志，诗意融于笔墨自然令人遐想无限如见梦中奇景的美丽。是不用描摹也不用预演，初遇之时便脱口而出衷心赞叹的美丽。是先祖将感悟化为诗文，将刹那间的意蕴与情境定格成永恒，世代传承口口相传的美丽。

      段俊平先生古意盎然的作品，总能以最直观的方式，以笔墨间的韵致展现出这种我们寻求且渴盼着的美，令我们触目倾倒，心神俱醉，在无限美妙的艺术世界中徜徉翩跹，立即领略到书法回归传统、回归于美的诸般妙处，从而发自内心地生出愉悦和庆幸来：通过书作而能叩问自身灵魂，感受书法家的美学灵思，进而升华寻求到独属于自己的审美灵魂，是何等的幸事!
