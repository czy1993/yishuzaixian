       大明宣德年制青花龙纹高足杯一对，外绘青花纹饰，构图简练，笔道流畅。主体纹饰为三爪行龙，龙体飞舞，极为灵活。专家对此高足杯评价非常高，收藏价值不可限量。

       清乾隆 铜胎掐丝珐埌彩三足香炉。此香炉以红铜作胎，采用的珐琅颜料主要是以石英、硼砂、长石等矿物质制成，含有较多杂质，也因此胎体上有细小的砂眼，掐丝工艺线条自然流畅、粗细均匀，图案富于变化，充分展示了制作匠人的工艺水平，是一件不可多得的大开门到代精品。

       光绪元宝福建省官局造，此钱币在闽省人当地习贯用小额币值流，如：库平一钱四分四厘、七分二厘及三分六厘，其主币库平七钱二分因面值大，不便当时流通，故铸额不多，留存至今的非常稀少，可谓中国机制银元的大珍品。

       元符通宝 有篆、楷、行三种字体，又有铜、铁两种材质。字迹端丽，面文从上而右而下而左旋读，钱背光而无文。此组钱币一枚为元符通宝真书小平样钱，一枚为元符通宝篆书折三母钱，元符通宝行书小平样钱书法造诣真是一流，钱文就是出自宋代文学大家苏东坡所写，行书元符通宝，直径2.4厘米，重4.2克。无论是从形制、材质、钱风气息、铸造风格上看都是不可多求的稀世珍品。而另一枚“元符通宝”篆书折三母钱同属上美精品，元符通宝篆书折三，钱体厚重，字口深峻，笔划清晰，边圆廓正，穿廓凸起，母尊之相，乃为母钱无疑。其历千年，秘藏地宫，终于挡不住当今天下大兴土木之扰，被催醒而再现世。又幸得母身之佑，抗锈殊强，故其品相仍基本完好，殊为难得矣。故这组钱币有着极高的收藏价值。
