同前夫强卖字画索贿千万 “最牛女处长”被判

来源：admin     发布时间：2018-09-29

  　证监会“最牛女处长”李志玲受贿案终审落判。9月28日，中国裁判文书网公布该案二审裁定：维持一审判处李志玲无期徒刑的判决。

　　裁定书显示，李志玲个人或伙同其前夫乔东方收受、索要财物价值数千万元 ，除了现金，涉案财物还包括奔驰汽车、浪琴手表等。

　　此外，二审认定，李志玲伙同乔东方以推销国画、油画、瓷器等艺术品为名，共同向某集团索要贿赂款共计3244万元。

　　收CEO送的奔驰车并索贿，辩称两人系情人

　　1973年生的李志玲，曾任证监会发行监管部审核二处主任科员、审核四处主任科员、助理调研员、副处长、处长、监管六处处长，主要负责公司首次公开发行股票、再融资申请的财务审核。

　　据微信公号“长安街知事”报道，李志玲因手握重权，加上作风“比较凶”，只愿见上市公司的董事长，被称为证监会“最牛女处长”。

　　法院认定的六起受贿或索贿事实，均和公司再融资有关。

　　例如，法院查明，2004年至2009年间，李志玲为浙江某公司首次公开发行股票和再融资提供帮助，2009年11月，她伙同乔东方共同收受时任该公司法定代表人的过某给予的800万元；2014年间，李志玲为某公司再融资提供帮助，收受某公司通过某资产公司业务总监郑某给予的浪琴牌手表2块。

　　此外，2007年至2014年间，李志玲为某集团下属公司再融资提供帮助，收受时任某集团首席执行官李某给予的奔驰牌汽车1辆，并向李某索要540万元。

　　关于收受奔驰车和向李某索贿，李志玲在一审判决后上诉称，她与李某系情人关系，双方之间的经济往来系民间借贷并非索贿，涉案奔驰车也不应认定为贿赂。

　　对此，法院认为，李志玲长期控制、支配涉案车辆，足以证明其具有收受该车的主观故意和客观行为。此外，她明知自己的职权能够制约李某任职的某集团下属公司再融资，向李某借款后拒不归还，应认定系以借为名索贿。

　　和前夫强卖字画，被认定索贿超三千万

　　裁定书显示，2015年6月19日，李志玲、乔东方在珠海度假时被抓获归案。次日，证监会发布消息称，因李志玲配偶违规买卖股票，决定对她作出行政开除处分，因涉嫌职务犯罪，李志玲被移送司法机关。

　　李志玲和乔东方曾是夫妻。根据裁定书，两人均称自2008年起就分居，但直至2014年3月15日才办理离婚手续。而在离婚前不久的2013年6月至7月，证监会纪委相继收到对李志玲配偶乔东方违规买卖股票、巨额财产来源不明的举报，当年7月31日，证监会纪委对李志玲进行了约谈。

　　二审裁定中，法院认定李志玲和乔东方共同犯罪的事实有两项，除了2009年11月两人共同收某公司法定代表人800万，另一项是在2007年至2014年间，李志玲为某集团的下属公司再融资提供帮助，其间伙同乔东方以推销国画、油画、瓷器等艺术品为名，共同向某集团索要贿赂款共计3244万元。

　　据公开报道，乔东方曾经营古董字画生意，曾有文章称，他所注册的公司是国内第一家私营拍卖企业，而其拍卖的物品多数是海外回流的，乔东方说他“是一个真正的收藏家”。

　　根据裁定书，李志玲当时是证监会负责某集团下属上市公司再融资财务审核的工作人员，她与乔东方接触该上市公司再融资负责人叶某后，乔东方成批量向该集团出售国画、油画等物品。

　　交易过程中，双方既不出具货品清单，也不出具收货凭证，双方没有书面合同，也不要求真伪鉴定。叶某作证称，送画后，李志玲、乔东方还向其催要钱款。

　　法院认定，涉案某集团并无主动购买作品的真实意愿，对艺术品的真伪和价值漠不关心，双方仅是将涉案艺术品作为输送利益的媒介和掩饰。因此，乔东方伙同李志玲在违背对方真实意愿的情况下，利用李志玲在证监会的职权，强行推销大批量的艺术品后不断追索货款的行为应认定为索贿，不适用《最高人民法院、最高人民检察院关于办理受贿刑事案件适用法律若干问题的意见》第一条第一款之规定，无需按照交易时当地市场价格与实际支付价格的差额计算受贿数额。

　　2018年9月20日，北京高院对李志玲案作出二审裁定，驳回李志玲的上诉，维持一审判决：李志玲犯受贿罪，判处无期徒刑，剥夺政治权利终身，并处没收个人全部财产。

　　澎湃新闻查询得知，2018年7月27日，北京高院亦对乔东方案作出二审裁定，维持一审以受贿罪判处乔东方有期徒刑15年的判决。
